# HealingGO
## About The Project
Application name : PAM-async-API

Aplikasi ini digunakan untuk menampilkan informasi orang. Informasi detail dapat dilihat apabila mengklik list orang.

### Daftar package yang digunakan

    * React Native
    * React Navigation
    * DateTimePicker
